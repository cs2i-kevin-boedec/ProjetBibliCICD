# Frontend - README

## Installation

Avant de lancer le projet, installez les dépendances avec la commande suivante :

```bash
npm install
```

## Lancement du projet

Une fois l'installation terminée, démarrez le serveur de développement avec :

```bash
ng serve
```

Le projet sera accessible à l'adresse suivante :

```
http://localhost:4200/
```
